Bootstrap 5 Directory Structure

css/
  • bootstrap.css
  • bootstrap.min.css
  • bootstrap.rtl.css
  • bootstrap-grid.css
  • bootstrap-utilities.css

Contains all compiled CSS stylesheets. Use bootstrap.min.css for production.

js/
  • bootstrap.js
  • bootstrap.min.js
  • bootstrap.bundle.js
  • bootstrap.bundle.min.js
  • bootstrap.esm.js

Contains JavaScript plugins. bundle versions include Popper.js.

icons/
  • bootstrap-icons.css
  • bootstrap-icons.woff
  • bootstrap-icons.woff2
  • fonts/

Contains icon font files. Over 1,800+ icons available as web fonts.
