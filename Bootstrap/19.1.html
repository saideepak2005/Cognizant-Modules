Exercise 19.1 - Bootstrap 5 Sass Setup

This page demonstrates the steps to set up a Bootstrap 5 project with Sass using npm.

Step 1: Initialize npm Project
npm init -y

Creates a package.json file in your project directory.

Step 2: Install Bootstrap and Sass
npm install bootstrap@5.3.3
npm install --save-dev sass

Installs Bootstrap source files and the Dart Sass compiler.

Step 3: Project File Structure
project/
├── node_modules/
├── scss/
│   ├── _variables.scss
│   └── main.scss
├── css/
│   └── main.css  (compiled)
├── index.html
└── package.json
Step 4: main.scss Content
@import "variables";
@import "../node_modules/bootstrap/scss/bootstrap";

Import your custom variables before Bootstrap so they override the defaults.

Step 5: Add Compile Script

In package.json scripts section:

"scripts": {
  "sass": "sass scss/main.scss css/main.css",
  "sass:watch": "sass --watch scss/main.scss css/main.css"
}

Run npm run sass:watch during development.

Step 6: Link Compiled CSS
<link href="css/main.css"
      rel="stylesheet">

Link the compiled CSS instead of the CDN in your HTML file.
