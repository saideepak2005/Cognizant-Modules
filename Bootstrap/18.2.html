Bootstrap Accordion

Collapsible content sections powered by Bootstrap JS.

Bootstrap 5 is the latest major version of the popular CSS framework. It includes a powerful grid system, utility classes, pre-built components, and JavaScript plugins to help build responsive, mobile-first websites quickly.

The accordion uses Bootstrap's Collapse JavaScript plugin. Each item has a button that toggles the visibility of its content panel. Using data-bs-parent ensures only one panel is open at a time.
