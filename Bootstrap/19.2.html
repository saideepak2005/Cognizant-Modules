Exercise 19.2 - Customizing via Sass Variables

In a real Sass project, override Bootstrap variables in _variables.scss before importing Bootstrap.

_variables.scss (Example)
// Override primary color
$primary: #6f42c1;

// Override border radius
$border-radius: 20px;
$border-radius-lg: 24px;
$border-radius-sm: 12px;

// Override font family
$font-family-base: 'Georgia', serif;

// Override spacing
$spacer: 1.2rem;

// Override theme colors map
$theme-colors: (
  "primary": $primary,
  "secondary": #6c757d,
  "success": #20c997,
  "danger": #e74c3c,
);
main.scss Import Order
// 1. Override variables FIRST
@import "variables";

// 2. Then import Bootstrap
@import "../node_modules/bootstrap/scss/functions";
@import "../node_modules/bootstrap/scss/variables";
@import "../node_modules/bootstrap/scss/mixins";
@import "../node_modules/bootstrap/scss/bootstrap";

// 3. Add your custom styles after
@import "custom";

Order matters: your variables must come before Bootstrap's own variable definitions.
